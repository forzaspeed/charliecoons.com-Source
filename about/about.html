About This Site

This site was written and built with completely free and open source programs. All assets were created in the vector graphics program Inkscape, and the basic HTML and CSS styling was done through Notepad++.

Design inspiration was taken from the 2000s UI aesthetic that used raster-based transparency and blur effects, known colloquially as "Frutiger Aero," named for the Windows Aero UI themes used in Windows Vista and Windows 7. I emulated this through transparent SVGs made in Inkscape, where I also created the abstract Aurora backgrounds. You can see these kinds of UI elements and backgrounds in both the aforementioned versions of Windows, along with Mac OS X and the PlayStation 3 "Cross Media Bar" interface.
